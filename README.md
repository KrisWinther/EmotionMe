# EmotionMe
## Трекер настроения для Android. Записывай своё эмоциональное состояние, отслеживай паттерны и получай мотивационные сообщения.

## Возможности

#### •	Запись настроения с выбором эмоции, места и погоды
#### •	Текстовые заметки к каждой записи
#### •	Статистика в виде столбчатых диаграмм за день / неделю / месяц
#### •	История заметок с фильтрацией по периоду
#### •	Ежедневные уведомления с настраиваемым временем
#### •	Регистрация и авторизация (локальная, без интернета)
#### •	Мотивационные сообщения на основе анализа настроения

## Технические требования

#### •	Android 8.0 (API 26) и выше
#### •	Kotlin 1.9+
#### •	Gradle with Version Catalogs (libs.versions.toml)

## Установка

#### 1. Клонируйте репозиторий: git clone https://github.com/KrisWinther/EmotionMe.git
#### 2. Откройте проект в Android Studio Arctic Fox или новее.
#### 3. Синхронизируйте Gradle и запустите на эмуляторе или устройстве.

## Структура проекта
